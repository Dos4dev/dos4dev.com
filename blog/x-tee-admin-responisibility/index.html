X-Tee system admin responsibility.

Posted at Jan 5, 2022

System admin is important role in X-Road system. There a few documents contain information about system admin responsobility:

  1. An official statute.

  2. X-Road documentation.

  3. Agreement between security server owner and RIA

To make it clearer I created a list of responsibilities:

  1. Archive the message log. The agreement with RIA is more precise about it.

  2. Setup the granted access policy rules.

  3. Ensute the execution of the requirements.

  4. Host servers in Estonia.

  5. Update the software

  6. Setup the communication with the Client Information Systems

  7. Manage the accessability of the ports and firewall configuration.
